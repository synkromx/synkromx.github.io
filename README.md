# synkro.mx

Sitio web oficial de **SYNKRO Tech** — Agencia Digital especializada en IA, redes sociales y automatización para PYMES.

## Estructura
```
synkro-site/
├── index.html          # Página principal
├── privacidad.html     # Aviso de privacidad (Meta compliance)
├── terminos.html       # Términos de servicio (Meta compliance)
├── datos.html          # Solicitud de eliminación de datos (Meta compliance)
├── CNAME               # Dominio personalizado para GitHub Pages
├── css/
│   └── style.css       # Estilos principales
├── js/
│   └── main.js         # JavaScript interactividad
└── assets/
    ├── logo-horizontal.png
    ├── logo-cuadrado.png
    └── favicon.png
```

## Deploy
Hospedado en **GitHub Pages** con dominio personalizado `synkro.mx` conectado desde Namecheap.
